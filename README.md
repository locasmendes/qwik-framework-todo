# qwik-app

Extensão para pomodoro de um aplicativo originalmente para ToDo

- Implementação utilizando Qwik do TodoMVC: https://todomvc.com/.
- Vite.js.
- Express.js.

## Development

### Install

```
npm install
```

### Development Build

```
npm run dev
```

### Production Build

```
npm run build
```
